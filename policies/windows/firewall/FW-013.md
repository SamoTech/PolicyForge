# FW-013 — Block Common Malware & Lateral Movement Ports

**ID:** FW-013  
**Category:** Windows Firewall / Lateral Movement Prevention  
**Risk Level:** 🔴 Critical  
**OS:** Windows 10+, Windows 11, Server 2016+  
**Source:** CIS Benchmark · CISA · Microsoft Security Baseline

---

## Description

A focused set of inbound block rules for ports commonly abused by malware, ransomware, and lateral movement frameworks. These are explicit deny rules that act as a secondary defense layer even if default policy drifts.

## PowerShell — Lateral Movement Port Block Rules

```powershell
$blockRules = @(
    # Port, Protocol, Name, Rationale
    @{Port=4444; Proto='TCP'; Name='Metasploit Default Meterpreter'; Note='Metasploit default handler'},
    @{Port=4445; Proto='TCP'; Name='Metasploit Alt'; Note='Common Metasploit variant'},
    @{Port=8443; Proto='TCP'; Name='Alt HTTPS C2'; Note='Common C2 framework alt port'},
    @{Port=1433; Proto='TCP'; Name='Block SQL Server Inbound'; Note='Restrict to DB servers only'},
    @{Port=1434; Proto='UDP'; Name='Block SQL Browser'; Note='SQL Browser enumeration'},
    @{Port=5985; Proto='TCP'; Name='Block WinRM HTTP'; Note='Use HTTPS only (FW-010)'},
    @{Port=23;   Proto='TCP'; Name='Block Telnet'; Note='Cleartext protocol'},
    @{Port=21;   Proto='TCP'; Name='Block FTP'; Note='Cleartext credentials'},
    @{Port=69;   Proto='UDP'; Name='Block TFTP'; Note='Often abused by implants'},
    @{Port=514;  Proto='UDP'; Name='Block Syslog Inbound'; Note='Prevent log injection'},
    @{Port=135;  Proto='TCP'; Name='Restrict RPC'; Note='Allow only from mgmt subnet'}
)

foreach ($rule in $blockRules) {
    New-NetFirewallRule `
        -DisplayName "PolicyForge: Block $($rule.Name)" `
        -Direction Inbound `
        -Protocol $rule.Proto `
        -LocalPort $rule.Port `
        -Action Block `
        -Profile Any `
        -Description $rule.Note `
        -Enabled True | Out-Null
    Write-Host "[+] Blocked: $($rule.Proto)/$($rule.Port) — $($rule.Name)"
}
```

## Ransomware Propagation Ports

```
Common ransomware lateral movement ports:
  TCP 445  (SMB)         → WannaCry, NotPetya, BlackMatter — FW-006
  TCP 3389 (RDP)         → Most ransomware manual ops — FW-005
  TCP 139  (NetBIOS)     → Legacy SMB — FW-004, FW-006
  TCP 135  (RPC/DCOM)    → WMI-based spread — FW-013
  TCP 1433 (SQL)         → Maze, REvil SQL propagation — FW-013
  TCP 5985 (WinRM)       → PS-Remoting spread — FW-010, FW-013
  UDP 137-138 (NetBIOS)  → LLMNR poisoning — FW-004

With full firewall baseline (FW-001 through FW-013):
  → Ransomware cannot spread via SMB between workstations
  → RDP brute force impossible from internet
  → WinRM lateral movement blocked from non-mgmt hosts
  → LLMNR/NBT-NS poisoning blocked
  → Metasploit default ports blocked
```
