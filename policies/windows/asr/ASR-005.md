# ASR-005 — Block Office Applications from Injecting Code into Other Processes

**ID:** ASR-005  
**Category:** Attack Surface Reduction / Defender ASR

## Description

Prevents Office applications from injecting code into other processes via techniques like `VirtualAllocEx`, `WriteProcessMemory`, or `CreateRemoteThread`. Process injection is a core technique for running shellcode, bypassing application controls, and hiding malicious activity inside legitimate processes.

## Impact

- ✅ Blocks process injection from Office applications
- ✅ Prevents shellcode injection via Office exploits
- ⚠️ Breaks screen readers and accessibility tools that hook Office processes
- ⚠️ Some Office add-ins use legitimate injection — test carefully
