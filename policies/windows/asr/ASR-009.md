## Description

Blocks attempts to read memory from LSASS (Local Security Authority Subsystem Service) — the process that stores NTLM hashes, Kerberos tickets, and other credentials in memory. This is the ASR-layer complement to CRED-003 (LSA PPL). Detects and blocks OpenProcess() + ReadProcessMemory() calls targeting lsass.exe, which is the core Mimikatz credential dumping technique.

## Impact

- ✅ Blocks Mimikatz sekurlsa::logonpasswords and similar tools
- ✅ Blocks ProcDump -ma lsass.exe credential dump technique
- ⚠️ May conflict with some EDR/AV products that legitimately read LSASS
- ℹ️ Complements but does not replace CRED-003 (LSA PPL) and CRED-004 (Credential Guard)
