## Description

Detects and blocks scripts that use obfuscation techniques — char/string encoding, base64-embedded code, XOR encoding, compressed blobs — commonly used by malware to evade static analysis. Covers PowerShell, JavaScript, VBScript, and other scripting engines.

## Impact

- ✅ Blocks heavily obfuscated malware scripts
- ✅ High-signal detection for attacker TTPs
- ⚠️ May block legitimate software that uses obfuscation for IP protection (licensing tools, some installers)
- ℹ️ Higher false positive rate — audit carefully before enforcing
