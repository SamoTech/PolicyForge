# ASR-006 — Block JavaScript and VBScript from Launching Downloaded Executable Content

**ID:** ASR-006  
**Category:** Attack Surface Reduction / Defender ASR  
**Risk Level:** 🔴 Critical  
**OS:** Windows 10 1709+, Windows 11  
**GUID:** `d3e037e1-3eb8-44c8-a917-57927947596d`  
**Source:** Microsoft Defender ASR Rules Reference


## Description

Blocks JavaScript (.js) and VBScript (.vbs) files from launching executables they download from the internet. This breaks the dropper pattern where a .js/.vbs attachment downloads and executes a payload binary. One of the most effective rules for blocking drive-by and email-based malware delivery.

## Impact

- ✅ Blocks .js/.vbs dropper patterns (massive reduction in ransomware delivery)
- ✅ Stops WSH (Windows Script Host) based downloaders
- ⚠️ May break legitimate .js/.vbs admin scripts that download tools
- ℹ️ Combine with CSCRIPT/WSCRIPT disable for maximum effect
