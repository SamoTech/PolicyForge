# ASR-012 — Block Office Communication Apps from Creating Child Processes

**ID:** ASR-012  
**Category:** Attack Surface Reduction / Office / Communication Apps  
**Risk Level:** 🔴 Critical

## Description

Blocks Microsoft Teams, Skype for Business, Lync, and other Office communication applications from spawning child processes. Communication apps are increasingly used as initial access vectors — malicious links or files in Teams chats can execute payloads. This rule stops the communication app → shell execution chain.

## Impact

- ✅ Blocks Teams/Skype-delivered malware execution chains
- ✅ Stops browser/file launches from communication apps that spawn malicious processes
- ⚠️ May block legitimate Teams integrations that spawn helper processes
- ⚠️ Some screen sharing tools may be affected
