# URA-001 — Deny SeDebugPrivilege to Non-Admins

**ID:** URA-001  
**Category:** User Rights Assignment / Privilege Control  
**Risk Level:** 🔴 Critical  
**OS:** Windows 10+, Windows 11, Server 2016+  
**Source:** CIS Benchmark Windows 11 v3.0 · DISA STIG · Microsoft Security Baseline

---

## Policy Path

```
Computer Configuration
  └─ Windows Settings
       └─ Security Settings
            └─ Local Policies
                 └─ User Rights Assignment
                      └─ Debug programs: Administrators
```

## Registry / secedit

```ini
; Security template (.inf)
[Privilege Rights]
SeDebugPrivilege = *S-1-5-32-544  ; Administrators group only
```

## PowerShell

```powershell
# Check current SeDebugPrivilege holders
Secedit /export /cfg C:\PolicyForge\current_rights.inf /quiet
Select-String "SeDebugPrivilege" C:\PolicyForge\current_rights.inf

# Apply via secedit template
$template = @"
[Unicode]
Unicode=yes
[Version]
signature="`$CHICAGO`$"
[Privilege Rights]
SeDebugPrivilege = *S-1-5-32-544
"@
$template | Out-File C:\PolicyForge\debug_priv.inf -Encoding Unicode
Secedit /configure /db C:\Windows\security\database\debug.sdb /cfg C:\PolicyForge\debug_priv.inf /quiet
```

## Description

`SeDebugPrivilege` allows a process to read/write memory of ANY other process, bypassing all object-level security. This is the privilege Mimikatz uses to read LSASS memory and extract credentials. By default, only Administrators should hold this right. The critical finding is when service accounts, developers, or regular users have been granted this privilege — intentionally or via broad group membership.

## Attack Scenarios

| Tool | Uses SeDebugPrivilege | Impact |
|---|---|---|
| **Mimikatz** `sekurlsa::logonpasswords` | ✅ Required | Dumps all cached credentials from LSASS |
| **ProcDump -ma lsass.exe** | ✅ Required | Creates LSASS dump for offline cracking |
| **Process Hacker** memory read | ✅ Required | Read secrets from any process memory |
| **WinPmem** | ✅ Required | Raw memory acquisition |
| Token impersonation attacks | ✅ Required | Steal tokens from higher-privileged processes |
