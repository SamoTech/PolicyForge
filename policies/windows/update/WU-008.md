# WU-008 — Disable Update Notifications to End Users

**ID:** WU-008  
**Category:** Windows Update / User Experience / Security  
**Risk Level:** 🟢 Low  
**OS:** Windows 10+, Windows 11  
**Source:** CIS Benchmark Windows 11 v3.0

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ Windows Update
                 └─ Manage end user experience
                      ├─ Display options for update notifications: 2 (Disable all notifications)
                      └─ Remove access to use all Windows Update features
                           └─ Enabled: 1 (Admins only) or 2 (Admins + users blocked from WU)
```

## Registry

```
HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate
  SetUpdateNotificationLevel = 2    (0=default, 1=disable restart, 2=disable all)
  DisableWindowsUpdateAccess = 1    (1=block WU UI access for non-admins)
  NoWindowsUpdate            = 0    (0=allow WU, 1=block entirely)
```

## PowerShell

```powershell
# Suppress update notifications
Set-ItemProperty -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate" `
    -Name "SetUpdateNotificationLevel" -Value 2

# Block user access to Windows Update settings
Set-ItemProperty -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate" `
    -Name "DisableWindowsUpdateAccess" -Value 1

# Verify
Get-ItemProperty "HKLM:\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate" |
    Select SetUpdateNotificationLevel, DisableWindowsUpdateAccess
```

## Description

Suppresses Windows Update notification popups and blocks non-admin users from accessing Windows Update settings. Notification suppression prevents users from being distracted by update prompts or, worse, clicking "Update now" during a critical work session. Blocking WU UI access prevents users from disabling updates, checking for updates outside the managed schedule, or installing unauthorized updates. IT retains full control.

## Impact

- ✅ No disruptive update notification popups
- ✅ Users cannot disable updates or change update settings
- ✅ Centralizes update management to IT
- ⚠️ Users cannot manually trigger updates when needed
- ℹ️ Help desk should have a process for urgent manual update requests
