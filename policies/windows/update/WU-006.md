# WU-006 — Disable Driver Updates via Windows Update

**ID:** WU-006  
**Category:** Windows Update / Driver Management  
**Risk Level:** 🟡 Medium  
**OS:** Windows 10+, Windows 11  
**Source:** CIS Benchmark Windows 11 v3.0 · Microsoft Security Baseline

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ Windows Update
                 └─ Manage updates offered from Windows Update
                      └─ Do not include drivers with Windows Updates
                           └─ Enabled
```

## Registry

```
HKLM\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate
  ExcludeWUDriversInQualityUpdate = 1    (1=exclude drivers, 0=include)
```

## PowerShell

```powershell
# Exclude drivers from Windows Update
Set-ItemProperty -Path "HKLM:\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate" `
    -Name "ExcludeWUDriversInQualityUpdate" -Value 1

# Verify
Get-ItemProperty "HKLM:\SOFTWARE\Policies\Microsoft\Windows\WindowsUpdate" |
    Select ExcludeWUDriversInQualityUpdate

# List installed drivers and their sources
Get-WindowsDriver -Online | Select Driver, ProviderName, Version, Date | Sort Date -Descending | Select -First 20

# Check for driver updates manually
pnputil /enum-drivers
```

## Description

Prevents Windows Update from automatically installing driver updates alongside security patches. While convenient, auto-installed drivers can break specialized hardware (graphics cards, audio interfaces, industrial controllers) or introduce untested firmware. In enterprise environments, drivers should be validated and deployed through SCCM/Intune or vendor-provided installers — not through Windows Update. This setting is especially important for workstations with specialized hardware or production equipment.

## Impact

- ✅ Prevents auto-driver updates from breaking specialized hardware
- ✅ Enterprise-controlled driver deployment via SCCM/Intune
- ✅ Avoids driver rollbacks after failed Windows Update installs
- ⚠️ Manual process required for driver updates — must have alternative deployment
- ⚠️ Some security-relevant driver updates won't auto-install
