# WU-010 — Enable Microsoft Update (Office + Other Microsoft Products)

**ID:** WU-010  
**Category:** Windows Update / Product Coverage  
**Risk Level:** 🟡 Medium  
**OS:** Windows 10+, Windows 11  
**Source:** CIS Benchmark Windows 11 v3.0

---

## PowerShell

```powershell
# Enable Microsoft Update (includes Office, .NET, Visual C++ Redistributables, etc.)
$ServiceManager = New-Object -ComObject Microsoft.Update.ServiceManager
$ServiceManager.AddService2("7971f918-a847-4430-9279-4a52d1efe18d", 7, "")
# GUID 7971f918... is the Microsoft Update service identifier

# Verify Microsoft Update is registered
$ServiceManager.Services | Select ServiceID, Name, IsRegisteredWithAU

# Alternative: Enable via registry
# Note: This requires the Microsoft Update client to be registered first
Get-WindowsUpdate -MicrosoftUpdate -NotInstalled | Select Title, Size, MsrcSeverity

# Check if Office updates are coming through WU
Get-HotFix | Where-Object { $_.Description -like "*Office*" } | Select -First 5
```

## Description

By default, Windows Update only covers Windows OS components. Enabling Microsoft Update extends coverage to all Microsoft products including Office, .NET Framework, Visual C++ Redistributables, SQL Server tools, and other Microsoft software. These components have their own CVEs and security patches — Office vulnerabilities are frequently exploited via malicious documents. Without Microsoft Update enabled, Office patches must be deployed through SCCM or manually, creating patch lag.

## Microsoft Update Coverage

| Product | Update Type | Frequency |
|---|---|---|
| Microsoft Office | Security + Feature | Monthly (Patch Tuesday) |
| .NET Framework | Security + Cumulative | Monthly |
| Visual C++ Redist | Security | As needed |
| SQL Server (tools) | Security | Monthly |
| Exchange (client) | Security | Monthly |
| Silverlight | Security | As needed |

## Impact

- ✅ Office security patches included in automated patching
- ✅ Single management point for all Microsoft software
- ✅ Eliminates common gap where OS is patched but Office isn't
- ⚠️ Increases update download size
- ℹ️ Use with WSUS for centralized approval of all product updates
