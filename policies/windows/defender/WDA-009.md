# WDA-009 — Configure Defender Scan Schedule & Types

**ID:** WDA-009  
**Category:** Windows Defender AV / Scan Configuration  
**Risk Level:** 🟡 Low  
**OS:** Windows 10+, Windows 11, Server 2016+  
**Source:** Microsoft Security Baseline

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ Microsoft Defender Antivirus
                 └─ Scan
                      ├─ Specify the scan type to use for a scheduled scan: Full Scan
                      ├─ Specify the day of the week to run a scheduled scan: Sunday (1)
                      ├─ Specify the time of day to run a scheduled scan: 120 (2:00 AM)
                      ├─ Turn on catch-up full scan: Enabled
                      ├─ Specify the maximum percentage of CPU usage during scan: 50
                      └─ Scan removable drives: Enabled
```

## Registry

```
HKLM\SOFTWARE\Policies\Microsoft\Windows Defender\Scan
  ScanParameters              = 2    (1=quick, 2=full)
  ScheduleDay                 = 1    (0=daily, 1=Sun, 2=Mon... 8=never)
  ScheduleTime                = 120  (minutes from midnight → 2:00 AM)
  DisableCatchupFullScan      = 0    (0=enable catchup if missed)
  AvgCPULoadFactor            = 50   (max 50% CPU during scan)
  DisableRemovableDriveScanning = 0  (scan USB drives)
  CheckForSignaturesBeforeRunningScan = 1  (update signatures before scanning)
```

## PowerShell

```powershell
# Configure scheduled scan
Set-MpPreference `
    -ScanParameters FullScan `
    -ScanScheduleDay Sunday `
    -ScanScheduleTime 02:00 `
    -DisableCatchupFullScan $false `
    -ScanAvgCPULoadFactor 50 `
    -DisableRemovableDriveScanning $false

# Trigger immediate full scan (for testing)
Start-MpScan -ScanType FullScan

# Check last scan results
Get-MpComputerStatus | Select `
    QuickScanStartTime, QuickScanEndTime,
    FullScanStartTime, FullScanEndTime,
    FullScanSignatureVersion
```

## Scan Type Guidance

| Scan Type | Coverage | Duration | When |
|---|---|---|---|
| **Quick** | Memory, startup, known malware paths | 1-5 min | Daily automatic |
| **Full** | Every file on disk | 1-4 hours | Weekly scheduled |
| **Custom** | Specific paths | Variable | On-demand |
| **Offline** | Pre-boot persistent malware | Reboot required | Suspected rootkit |
