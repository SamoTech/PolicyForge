# WDA-010 — Enable Defender Exploit Protection (EMET Successor)

**ID:** WDA-010  
**Category:** Windows Defender / Exploit Protection  
**Risk Level:** 🔴 Critical  
**OS:** Windows 10 1709+, Windows 11  
**Source:** Microsoft Security Baseline · CIS Benchmark

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ Microsoft Defender Exploit Guard
                 └─ Exploit Protection
                      └─ Use a common set of exploit protection settings:
                           Enabled → XML file path
```

## PowerShell — Apply Exploit Protection

```powershell
# Export current settings
Get-ProcessMitigation -System | Format-List

# Apply system-level mitigations
# CFG (Control Flow Guard) — enabled by default on modern systems
# DEP (Data Execution Prevention) — always on
# ASLR (Address Space Layout Randomization)
Set-ProcessMitigation -System -Enable ASLR, ForceRelocateImages, BottomUp, HighEntropy

# Export to XML for GPO deployment
Get-ProcessMitigation -RegistryConfigFilePath C:\PolicyForge\ExploitProtection.xml

# Apply XML via GPO or PowerShell
Set-ProcessMitigation -PolicyFilePath C:\PolicyForge\ExploitProtection.xml

# Per-process: harden high-risk processes
$highRisk = @("winword.exe", "excel.exe", "powerpnt.exe", "outlook.exe", "acrobat.exe")
$highRisk | ForEach-Object {
    Set-ProcessMitigation -Name $_ -Enable DEP, SEHOP, ASLR, CFG, ExtensionPointDisable
}
```

## System-Level Mitigations

| Mitigation | Blocks |
|---|---|
| **DEP** | Code execution from data pages (heap/stack spray) |
| **ASLR + HighEntropy** | Predictable memory layout exploitation |
| **CFG** | Return-oriented programming (ROP chains) |
| **SEHOP** | SEH (Structured Exception Handler) overwrite attacks |
| **BottomUp ASLR** | Stack-based exploitation |

## Per-Process Hardening

```powershell
# Additional hardening for Office and browser processes
Set-ProcessMitigation -Name "winword.exe" `
    -Enable DEP, SEHOP, ASLR, CFG, ExtensionPointDisable, `
             DisableNonSystemFonts, BlockRemoteImageLoads
```
