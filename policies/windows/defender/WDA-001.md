# WDA-001 — Enable Real-Time Protection (All Components)

**ID:** WDA-001  
**Category:** Windows Defender AV / Real-Time Protection  
**Risk Level:** 🔴 Critical  
**OS:** Windows 10+, Windows 11, Server 2016+  
**Source:** CIS Benchmark Windows 11 v3.0 · Microsoft Security Baseline

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ Microsoft Defender Antivirus
                 └─ Real-time Protection
                      ├─ Turn off real-time protection: Disabled
                      ├─ Turn on behavior monitoring: Enabled
                      ├─ Scan all downloaded files and attachments: Enabled
                      ├─ Monitor file and program activity on your computer: Enabled
                      ├─ Turn on process scanning whenever real-time protection is enabled: Enabled
                      └─ Turn on raw volume write notifications: Enabled
```

## Registry

```
HKLM\SOFTWARE\Policies\Microsoft\Windows Defender\Real-Time Protection
  DisableRealtimeMonitoring          = 0   (must be 0 = enabled)
  DisableBehaviorMonitoring          = 0
  DisableOnAccessProtection          = 0
  DisableScanOnRealtimeEnable        = 0
  DisableIOAVProtection              = 0   (scan downloads)
  RealtimeScanDirection              = 0   (0=both, 1=incoming, 2=outgoing)
  DisableRawWriteNotification        = 0
```

## PowerShell

```powershell
# Verify all real-time protection components
Get-MpPreference | Select `
    DisableRealtimeMonitoring,
    DisableBehaviorMonitoring,
    DisableOnAccessProtection,
    DisableIOAVProtection,
    RealTimeScanDirection

# Check current protection status
Get-MpComputerStatus | Select `
    AMRunningMode,
    RealTimeProtectionEnabled,
    BehaviorMonitorEnabled,
    IoavProtectionEnabled,
    OnAccessProtectionEnabled
```

## Description

Enables all components of Windows Defender real-time protection. Each component addresses a specific attack vector:

| Component | Blocks |
|---|---|
| Real-time monitoring | File creation/modification by malware |
| Behavior monitoring | Suspicious process behavior patterns |
| On-access protection | File execution before it runs |
| IOAV protection | Downloaded files and email attachments |
| Raw write notifications | Ransomware raw disk writes (MBR attacks) |
| Process scanning | Malicious code injected into new processes |

## Impact

- ✅ All malware vectors covered at file/process/behavior level
- ✅ Ransomware raw disk write attempts detected
- ⚠️ Slight CPU overhead on high-I/O workloads (servers)
- ℹ️ On servers: configure server-specific exclusions (URA-004) to balance performance
