# WDA-012 — Enable IOAV & Download Scanning

**ID:** WDA-012  
**Category:** Windows Defender AV / Download Protection  
**Risk Level:** 🔴 Critical  
**OS:** Windows 10+, Windows 11  
**Source:** CIS Benchmark Windows 11 v3.0

---

## Registry

```
HKLM\SOFTWARE\Policies\Microsoft\Windows Defender\Real-Time Protection
  DisableIOAVProtection = 0   (0=scan downloads, 1=disabled)

HKLM\SOFTWARE\Policies\Microsoft\Windows Defender\Scan
  DisableArchiveScanning     = 0   (scan inside ZIP, RAR, 7z)
  DisableEmailScanning       = 0   (scan email attachments)
  DisableHeuristics          = 0   (enable heuristic detection)
  DisablePackedExeScanning   = 0   (scan packed/UPX executables)
```

## Description

IOAV (Interfaces of Anti-Virus) protection scans all files immediately upon download before they are written to disk. Disabling archive scanning is a common attacker trick — malware delivered in ZIP files avoids detection if archives aren't unpacked and scanned. Email attachment scanning ensures that Outlook and other email clients trigger Defender scans on all attachments before opening.

## Common Evasion Techniques (Blocked)

| Evasion | Technique | Blocked By |
|---|---|---|
| ZIP delivery | Malware in password-protected ZIP | Archive scanning |
| Packed executables | UPX/custom packer to evade hash detection | Packed exe scanning |
| Email attachment | .docm with macro sent via email | Email scanning |
| Heuristic bypass | Slightly modified known malware | Heuristics enabled |
