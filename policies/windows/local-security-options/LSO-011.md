# LSO-011 — System Objects: Strengthen Default Permissions

**ID:** LSO-011  
**Category:** Local Security Options / Object Security  
**Risk Level:** 🟠 High  
**OS:** Windows 10+, Windows 11, Server 2016+  
**Source:** CIS Benchmark Windows 11 v3.0

---

## Policy Path

```
Computer Configuration
  └─ Windows Settings
       └─ Security Settings
            └─ Local Policies
                 └─ Security Options
                      ├─ System objects: Require case insensitivity for non-Windows subsystems: Enabled
                      ├─ System objects: Strengthen default permissions of internal system objects
                           (e.g. Symbolic links): Enabled
                      └─ System cryptography: Use FIPS compliant algorithms for encryption,
                           hashing, and signing: context-dependent
```

## Registry

```
HKLM\SYSTEM\CurrentControlSet\Control\Session Manager\Kernel
  ObCaseInsensitive      = 1   ; Case-insensitive object names

HKLM\SYSTEM\CurrentControlSet\Control\Session Manager
  ProtectionMode         = 1   ; Strengthen object permissions

HKLM\SYSTEM\CurrentControlSet\Control\Lsa
  FIPSAlgorithmPolicy    = 0   ; 1=FIPS mode (may break applications)
```

## Symbolic Link Hardening

```
Symbolic link attack without ProtectionMode = 1:
  Low-privilege process creates symbolic link in temp directory
  → Points to privileged system file
  → Privileged process follows symlink and overwrites target
  → Privilege escalation via symlink race condition
  (Windows CWE-59: Improper Link Resolution)

With ProtectionMode = 1:
  Object Manager enforces stricter ACLs on named objects
  → Reduces attack surface for object squatting and symlink attacks
```

## FIPS Mode Note

FIPS mode (FIPSAlgorithmPolicy = 1) enforces FIPS 140-2 approved algorithms. This **breaks** many common applications:
- RDP NLA may fail on some configurations
- Some .NET applications use non-FIPS algorithms
- Wi-Fi WPA2-Personal may not work

For most enterprises: **leave FIPS mode off** unless required by regulation (FISMA, FedRAMP).
