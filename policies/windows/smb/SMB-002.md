## PowerShell

```powershell
# Disable SMB compression (client)
Set-SmbClientConfiguration -DisableCompression $true -Force

# Disable SMB compression (server)
Set-SmbServerConfiguration -DisableCompression $true -Force

# Verify
Get-SmbClientConfiguration | Select DisableCompression
```

## Description

Disables SMB compression to mitigate exploitation of compression-related vulnerabilities (CVE-2020-0796 "SMBGhost" class). SMB compression was introduced in SMB 3.1.1 (Windows 10 2004) and has been a source of critical pre-auth RCE vulnerabilities.

## Impact

- ✅ Eliminates attack surface from SMB compression vulnerabilities
- ✅ Mitigates future compression-based exploit chains
- ⚠️ Slightly increases bandwidth usage for large file transfers over slow WAN links
- ℹ️ Negligible impact in LAN environments

## Use Cases

- Security hardening where bandwidth is not a constraint
- Environments with publicly accessible SMB shares
- Post-incident hardening after SMBGhost-class vulnerability disclosure
