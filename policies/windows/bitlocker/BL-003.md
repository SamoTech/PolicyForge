# BL-003 — Set Minimum BitLocker PIN Length (6+ digits)

**ID:** BL-003  
**Category:** BitLocker / PIN Policy  
**Risk Level:** 🟠 Medium  
**OS:** Windows 10 1511+, Windows 11  
**Source:** Microsoft BitLocker ADMX

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ BitLocker Drive Encryption
                 └─ Operating System Drives
                      └─ Configure minimum PIN length for startup
                           ├─ Enabled
                           └─ Minimum characters: 6
```

## Registry

```
Key:   HKLM\SOFTWARE\Policies\Microsoft\FVE
Value: MinimumPIN
Type:  REG_DWORD
Data:  6  (range: 4–20)
```

## PowerShell

```powershell
# Set via registry (requires admin)
Set-ItemProperty -Path "HKLM:\SOFTWARE\Policies\Microsoft\FVE" `
                 -Name "MinimumPIN" -Value 6 -Type DWord

# Check current BitLocker PIN length requirement
Get-ItemProperty -Path "HKLM:\SOFTWARE\Policies\Microsoft\FVE" -Name "MinimumPIN"
```

## Description

Enforces a minimum PIN length for BitLocker pre-boot authentication. The Windows default minimum is 4 digits, which provides 10,000 combinations. Setting 6 digits provides 1,000,000 combinations; 8 digits provides 100,000,000. Combined with TPM lockout (auto-locks after N wrong attempts), longer PINs provide strong brute-force resistance.

## Impact

- ✅ Dramatically improves brute-force resistance
- ✅ 6-digit minimum: 100× more combinations than 4-digit default
- ⚠️ Users must set/change PINs to meet the new minimum
- ℹ️ 8 digits recommended for high-security environments
