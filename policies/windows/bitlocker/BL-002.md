# BL-002 — BitLocker Encryption Algorithm: XTS-AES 256-bit

**ID:** BL-002  
**Category:** BitLocker / Encryption Strength  
**Risk Level:** 🔴 Critical  
**OS:** Windows 10 1511+, Windows 11  
**Source:** CIS Benchmark Windows 11 v3.0 · DISA STIG · NIST

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ BitLocker Drive Encryption
                 ├─ Choose drive encryption method and cipher strength
                 │    (Windows 10 [Version 1511] and later):
                 │    Enabled
                 │    ├─ OS drives: XTS-AES 256-bit
                 │    ├─ Fixed data drives: XTS-AES 256-bit
                 │    └─ Removable data drives: AES-CBC 256-bit
                 └─ (Older versions): AES 256-bit
```

## Registry

```
HKLM\SOFTWARE\Policies\Microsoft\FVE
  EncryptionMethodWithXtsOs          = 7   ; XTS-AES 256 for OS drive
  EncryptionMethodWithXtsFdv         = 7   ; XTS-AES 256 for fixed drives
  EncryptionMethodWithXtsRdv         = 6   ; AES-CBC 256 for removable (compat)
  ; Values: 3=AES-CBC 128, 4=AES-CBC 256, 6=XTS-AES 128, 7=XTS-AES 256
```

## Encryption Method Comparison

| Method | Key Size | Mode | Sector-Level | Use Case |
|---|---|---|---|---|
| AES-CBC 128 | 128-bit | CBC | ❌ | Legacy compat only |
| AES-CBC 256 | 256-bit | CBC | ❌ | Removable drives (cross-compat) |
| XTS-AES 128 | 128-bit | XTS | ✅ | Acceptable |
| **XTS-AES 256** | **256-bit** | **XTS** | **✅** | **Recommended (OS + Fixed)** |

## Why XTS over CBC?

```
AES-CBC weakness:
  Each sector uses same IV derivation
  → Watermarking attack: known plaintext reveals patterns
  → Cipher block chaining → sector corruption propagates

XTS (XEX-based Tweaked CodeBook with ciphertext Stealing):
  → Each 512-byte sector has unique tweak value
  → Bit flip in one sector does NOT propagate
  → No watermarking attack possible
  → IEEE P1619 standard for disk encryption
  → NIST SP 800-38E approved

Note: AES-CBC 256 still required for removable drives
      that must be read on older Windows systems (Win 7/8)
      XTS is only supported on Windows 10+
```
