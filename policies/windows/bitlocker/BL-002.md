# BL-002 — Require TPM 2.0 + PIN for BitLocker Pre-Boot Authentication

**ID:** BL-002  
**Category:** BitLocker / Pre-Boot Authentication  
**Risk Level:** 🔴 Critical  
**OS:** Windows 10 1511+, Windows 11  
**Source:** Microsoft BitLocker ADMX · DISA STIG WN11

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ BitLocker Drive Encryption
                 └─ Operating System Drives
                      └─ Require additional authentication at startup
                           ├─ Enabled
                           ├─ Allow BitLocker without a compatible TPM: Disabled
                           ├─ Configure TPM startup: Do not allow TPM
                           ├─ Configure TPM startup PIN: Require startup PIN with TPM
                           ├─ Configure TPM startup key: Do not allow startup key with TPM
                           └─ Configure TPM startup key and PIN: Do not allow startup key and PIN
```

## Registry

```
Key:   HKLM\SOFTWARE\Policies\Microsoft\FVE
Value: UseAdvancedStartup        = 1 (DWORD) — Enable advanced startup
Value: EnableBDEWithNoTPM        = 0 (DWORD) — Disallow no-TPM
Value: UseTPM                    = 0 (DWORD) — Do not allow TPM only
Value: UseTPMPIN                 = 1 (DWORD) — Require TPM+PIN
Value: UseTPMKey                 = 0 (DWORD) — No startup key
Value: UseTPMKeyPIN              = 0 (DWORD) — No key+PIN combo
```

## PowerShell

```powershell
# Set BitLocker to require TPM + PIN
$SecurePin = Read-Host -AsSecureString "Enter BitLocker PIN"
Enable-BitLocker -MountPoint "C:" `
                 -EncryptionMethod XtsAes256 `
                 -TpmAndPinProtector `
                 -Pin $SecurePin

# If BitLocker already enabled with TPM-only, add PIN:
Add-BitLockerKeyProtector -MountPoint "C:" `
                          -TpmAndPinProtector `
                          -Pin $SecurePin

# Remove old TPM-only protector
$tpmProtector = (Get-BitLockerVolume -MountPoint "C:").KeyProtector | 
                Where-Object { $_.KeyProtectorType -eq "Tpm" }
Remove-BitLockerKeyProtector -MountPoint "C:" -KeyProtectorId $tpmProtector.KeyProtectorId
```

## Description

Requires a user-defined PIN at every boot in addition to TPM verification. TPM-only BitLocker (default) is vulnerable to "evil maid" attacks where an attacker with physical access can extract the volume master key from the TPM using tools like PCILeech or cold boot attacks. Adding a PIN means the attacker needs both physical access AND knowledge of the PIN to decrypt the drive.

## Impact

- ✅ Eliminates TPM-only BitLocker cold boot / DMA attack vulnerability
- ✅ Requires attacker to know PIN AND have physical device
- ⚠️ Unattended/remote boot no longer possible — user must physically enter PIN
- ⚠️ Lost PIN with no recovery key = permanent data loss
- ℹ️ Use 6+ digit PIN; avoid simple sequences (1234, 0000)
- ℹ️ Network Unlock (BL-007) can bypass PIN requirement on trusted networks
