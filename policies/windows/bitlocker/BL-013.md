# BL-013 — BitLocker Compliance Monitoring & Reporting

**ID:** BL-013  
**Category:** BitLocker / Compliance Monitoring  
**Risk Level:** 🟠 High  
**OS:** Windows 10+, Windows 11, Server 2016+  
**Source:** Microsoft MDE · NIST 800-53

---

## PowerShell — Fleet-Wide BitLocker Status Report

```powershell
# Run against all domain computers
$results = @()
$computers = Get-ADComputer -Filter * -Properties OperatingSystem |
    Where-Object { $_.OperatingSystem -like "*Windows*" }

foreach ($computer in $computers) {
    try {
        $vol = Invoke-Command -ComputerName $computer.Name -ScriptBlock {
            $v = Get-BitLockerVolume -MountPoint C: -ErrorAction Stop
            [PSCustomObject]@{
                Computer           = $env:COMPUTERNAME
                VolumeStatus       = $v.VolumeStatus
                ProtectionStatus   = $v.ProtectionStatus
                EncryptionMethod   = $v.EncryptionMethod
                EncryptionPct      = $v.EncryptionPercentage
                ProtectorTypes     = ($v.KeyProtector.KeyProtectorType -join ', ')
                HasRecoveryKey     = ($v.KeyProtector.KeyProtectorType -contains 'RecoveryPassword')
                HasTPMProtector    = ($v.KeyProtector.KeyProtectorType -contains 'Tpm') -or
                                     ($v.KeyProtector.KeyProtectorType -contains 'TpmPin')
            }
        } -ErrorAction Stop
        $results += $vol
    } catch {
        $results += [PSCustomObject]@{
            Computer = $computer.Name
            VolumeStatus = "ERROR: $_"
        }
    }
}

# Output report
$results | Export-Csv C:\PolicyForge\BitLocker-Report-$(Get-Date -Format yyyyMMdd).csv -NoTypeInformation

# Highlight non-compliant
$nonCompliant = $results | Where-Object {
    $_.VolumeStatus -ne 'FullyEncrypted' -or
    $_.ProtectionStatus -ne 'On' -or
    $_.EncryptionMethod -notmatch '256'
}

Write-Host "`n=== NON-COMPLIANT MACHINES ==="
$nonCompliant | Format-Table Computer, VolumeStatus, ProtectionStatus, EncryptionMethod
```

## BitLocker Compliance Checklist

```
✅ VolumeStatus: FullyEncrypted (not EncryptionInProgress, FullyDecrypted)
✅ ProtectionStatus: On (not Off = suspended)
✅ EncryptionMethod: XtsAes256 (not AesCbc128, AesCbc256, XtsAes128)
✅ KeyProtector includes: TpmPin (laptops) or Tpm+NetworkUnlock (desktops)
✅ KeyProtector includes: RecoveryPassword (for helpdesk/recovery)
✅ Recovery key backed up to AD DS or Entra ID
✅ S3 sleep disabled or Kernel DMA Protection enabled
```

## BitLocker Status Codes

| VolumeStatus | Meaning | Action |
|---|---|---|
| FullyEncrypted | ✅ Complete | Verify ProtectionStatus = On |
| EncryptionInProgress | ⏳ Encrypting | Monitor to completion |
| FullyDecrypted | ❌ Not encrypted | Enable BitLocker immediately |
| DecryptionInProgress | ⚠️ Being decrypted | Investigate — who authorized? |
| EncryptionPaused | ⚠️ Stalled | Resume encryption |
| Unknown | ❌ Error | Investigate BitLocker service |
