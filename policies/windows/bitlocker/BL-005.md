# BL-005 — Require BitLocker on Fixed Data Drives

**ID:** BL-005  
**Category:** BitLocker / Fixed Data Drives  
**Risk Level:** 🔴 Critical  
**OS:** Windows 10 1511+, Windows 11  
**Source:** Microsoft BitLocker ADMX · CIS Benchmark

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ BitLocker Drive Encryption
                 └─ Fixed Data Drives
                      ├─ Deny write access to fixed drives not protected by BitLocker → Enabled
                      └─ Choose drive encryption method and cipher strength
                           └─ Fixed data drives: XTS-AES 256-bit
```

## Registry

```
Key:   HKLM\SOFTWARE\Policies\Microsoft\FVE
Value: FDVDenyWriteAccess    = 1 (DWORD) — Deny write to unencrypted fixed drives
Value: EncryptionMethodWithXtsFdv = 7 (DWORD) — XTS-AES 256
```

## PowerShell

```powershell
# Enable BitLocker on a fixed data drive (D:)
Enable-BitLocker -MountPoint "D:" `
                 -EncryptionMethod XtsAes256 `
                 -RecoveryPasswordProtector

# Auto-unlock fixed drive after OS drive is unlocked
Enable-BitLockerAutoUnlock -MountPoint "D:"

# Check status of all volumes
Get-BitLockerVolume | Select MountPoint, VolumeStatus, EncryptionMethod, AutoUnlockEnabled
```

## Intune CSP

```
OMA-URI: ./Device/Vendor/MSFT/BitLocker/FixedDrivesRequireEncryption
Type:    String
Value:   <enabled/>
```

## Description

Requires BitLocker encryption on all fixed (internal) data drives and denies write access to unencrypted fixed drives. Without this policy, data stored on secondary partitions (D:, E:) remains unencrypted even when the OS drive (C:) is BitLocker-protected. This closes a common security gap where sensitive files stored on non-OS partitions are exposed.

## Impact

- ✅ Encrypts all internal data drives, not just OS drive
- ✅ Denies write access to unencrypted fixed drives (enforced)
- ⚠️ Existing unencrypted fixed drives become read-only until encrypted
- ℹ️ Use Auto-unlock for seamless experience after OS drive unlock
