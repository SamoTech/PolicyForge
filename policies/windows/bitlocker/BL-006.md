# BL-006 — Require BitLocker on Removable Drives (USB)

**ID:** BL-006  
**Category:** BitLocker / Removable Drives / BitLocker To Go  
**Risk Level:** 🔴 Critical  
**OS:** Windows 10 1511+, Windows 11  
**Source:** Microsoft BitLocker ADMX · CIS Benchmark

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ BitLocker Drive Encryption
                 └─ Removable Data Drives
                      ├─ Deny write access to removable drives not protected by BitLocker → Enabled
                      ├─ Do not allow write access to devices configured in another organization: Enabled
                      └─ Choose drive encryption method (Removable): AES-CBC 256 or XTS-AES 128
```

## Registry

```
Key:   HKLM\SOFTWARE\Policies\Microsoft\FVE
Value: RDVDenyWriteAccess         = 1 (DWORD) — Deny write to unencrypted USB
Value: RDVDenyCrossOrg            = 1 (DWORD) — Deny write to non-org drives
Value: EncryptionMethodWithXtsRdv = 4 (DWORD) — AES-CBC 256 (compatible with older Windows)
```

## PowerShell

```powershell
# Enable BitLocker To Go on USB drive (E:)
Enable-BitLocker -MountPoint "E:" `
                 -EncryptionMethod AesCbc256 `
                 -Password (Read-Host -AsSecureString "Set USB Password")

# Check removable drive status
Get-BitLockerVolume | Where-Object { $_.VolumeType -eq "Data" }
```

## Intune CSP

```
OMA-URI: ./Device/Vendor/MSFT/BitLocker/RemovableDrivesRequireEncryption
Type:    String

```

## Description

Forces BitLocker To Go encryption on all USB/removable drives before allowing write access. Without this, employees can copy sensitive files to unencrypted USB drives and walk out with plaintext data. This policy blocks write access to any USB drive that isn't BitLocker-encrypted, effectively preventing unauthorized data exfiltration via USB.

## Impact

- ✅ Blocks sensitive data being written to unencrypted USB drives
- ✅ Major DLP (Data Loss Prevention) control
- ⚠️ Users can still read from unencrypted USB drives (read-only)
- ⚠️ USB drives must be formatted with BitLocker To Go before use
- ℹ️ BitLocker To Go drives are readable on Windows 7+ without additional software

## Use Cases

- Healthcare (HIPAA data on USB)
- Legal firms (client data portability)
- Government contractor endpoints
- Any environment with strict data exfiltration controls
