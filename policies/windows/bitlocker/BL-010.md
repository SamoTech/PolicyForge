# BL-010 — BitLocker Startup Key & USB Protectors

**ID:** BL-010  
**Category:** BitLocker / Startup Key Management  
**Risk Level:** 🟡 Medium  
**OS:** Windows 10+, Windows 11  
**Source:** Microsoft BitLocker Documentation

---

## Description

Startup key protectors store the BitLocker unlock key on a USB drive that must be present at boot. This is an alternative to TPM+PIN for systems without a TPM chip, and can be combined with TPM for a two-factor approach.

## Use Cases

| Scenario | Recommended Protector |
|---|---|
| Laptop with TPM 2.0 | TPM + PIN (BL-001) |
| Desktop with TPM 2.0 + corporate network | TPM + Network Unlock (BL-007) |
| System WITHOUT TPM | USB startup key (this policy) |
| Air-gapped workstation, high security | TPM + PIN + USB key (3-factor) |

## PowerShell

```powershell
# Enable BitLocker with USB startup key (no TPM systems)
Enable-BitLocker -MountPoint C: `
    -EncryptionMethod XtsAes256 `
    -StartupKeyPath E: `
    -StartupKeyProtector

# Add USB key to existing TPM-protected drive (2-factor)
Add-BitLockerKeyProtector -MountPoint C: `
    -StartupKeyPath E: `
    -StartupKeyProtector

# Verify protectors
(Get-BitLockerVolume C:).KeyProtector
```

## USB Key Security Rules

```
✔ Store USB key separately from device (different physical location)
✔ Label USB key only with recovery ID (no machine name/owner)
✔ Keep a backup USB key in secure storage (safe, locked cabinet)
✔ Replace USB key if lost/stolen and rotate BitLocker key
❌ Never leave USB key in the machine permanently
❌ Never store USB key with the laptop bag
```
