# BL-004 — Enable Enhanced BitLocker PIN (Allow Alphanumeric)

**ID:** BL-004  
**Category:** BitLocker / PIN Policy  
**Risk Level:** 🟠 Medium  
**OS:** Windows 10 1511+, Windows 11  
**Source:** Microsoft BitLocker ADMX · CIS Benchmark

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ BitLocker Drive Encryption
                 └─ Operating System Drives
                      └─ Allow enhanced PINs for startup
                           └─ Enabled
```

## Registry

```
Key:   HKLM\SOFTWARE\Policies\Microsoft\FVE
Value: UseEnhancedPin
Type:  REG_DWORD
Data:  1
```

## PowerShell

```powershell
Set-ItemProperty -Path "HKLM:\SOFTWARE\Policies\Microsoft\FVE" `
                 -Name "UseEnhancedPin" -Value 1 -Type DWord
```

## Description

Allows BitLocker PINs to include uppercase letters, lowercase letters, numbers, spaces, and symbols — not just numeric digits. A 6-character alphanumeric PIN with mixed case provides billions more combinations than a 6-digit numeric PIN. Note: enhanced PINs are entered in the pre-boot environment which may have limited keyboard support on some hardware.

## Impact

- ✅ Exponentially larger PIN keyspace
- ✅ Supports passphrase-style authentication
- ⚠️ Pre-boot keyboard may not support all special characters (test on target hardware)
- ⚠️ Non-US keyboard layouts may cause issues in pre-boot
- ℹ️ Test on all target hardware models before deploying
