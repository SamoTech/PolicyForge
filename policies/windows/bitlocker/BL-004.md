# BL-004 — BitLocker for Fixed Data Drives

**ID:** BL-004  
**Category:** BitLocker / Fixed Data Drives  
**Risk Level:** 🟠 High  
**OS:** Windows 10+, Windows 11, Server 2016+  
**Source:** CIS Benchmark Windows 11 v3.0 · Microsoft Security Baseline

---

## Policy Path

```
Computer Configuration
  └─ Administrative Templates
       └─ Windows Components
            └─ BitLocker Drive Encryption
                 └─ Fixed Data Drives
                      ├─ Deny write access to fixed drives not protected by BitLocker: Enabled
                      ├─ Choose how BitLocker-protected fixed drives can be recovered: Enabled
                      │    └─ Save to AD DS: Enabled
                      └─ Configure use of smart cards on fixed data drives: Enabled
```

## Registry

```
HKLM\SOFTWARE\Policies\Microsoft\FVE
  FDVDenyWriteAccess          = 1   ; Deny write to unencrypted fixed drives
  FDVRecovery                 = 1   ; Enable recovery
  FDVActiveDirectoryBackup    = 1   ; Backup recovery to AD
  FDVRequireActiveDirectoryBackup = 1
```

## PowerShell

```powershell
# Encrypt all unencrypted fixed drives
Get-BitLockerVolume | Where-Object {
    $_.VolumeType -eq 'Data' -and $_.VolumeStatus -ne 'FullyEncrypted'
} | ForEach-Object {
    Enable-BitLocker -MountPoint $_.MountPoint `
        -EncryptionMethod XtsAes256 `
        -RecoveryPasswordProtector
    # Backup recovery key to AD
    $id = (Get-BitLockerVolume $_.MountPoint).KeyProtector |
        Where-Object {$_.KeyProtectorType -eq 'RecoveryPassword'} |
        Select-Object -ExpandProperty KeyProtectorId
    Backup-BitLockerKeyProtector -MountPoint $_.MountPoint -KeyProtectorId $id
}
```

## Why Fixed Data Drives Need Separate Encryption

```
OS drive encrypted (C:) but D: data drive unencrypted:
  Attacker removes machine from domain
  → Boots with USB live OS
  → C: locked by BitLocker
  → D: fully readable (no encryption)
  → All data on D: extracted

With FDVDenyWriteAccess = 1:
  Windows refuses to write to any unencrypted fixed drive
  → Forces encryption of all internal data drives
  → No unencrypted data partition possible
```
