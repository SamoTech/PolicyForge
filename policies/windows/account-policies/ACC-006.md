# ACC-006 — Account Lockout Threshold

**ID:** ACC-006  
**Category:** Account Policies / Account Lockout  
**Risk Level:** 🔴 Critical  
**OS:** Windows 10+, Windows 11, Server 2016+  
**Source:** CIS Benchmark Windows 11 v3.0 · DISA STIG · Microsoft Security Baseline

---

## Policy Path

```
Computer Configuration
  └─ Windows Settings
       └─ Security Settings
            └─ Account Policies
                 └─ Account Lockout Policy
                      └─ Account lockout threshold: 5
```

## secedit

```ini
[System Access]
LockoutBadCount = 5
```

## PowerShell

```powershell
# Check current lockout policy
net accounts

# Get domain lockout policy
Get-ADDefaultDomainPasswordPolicy | Select LockoutThreshold, LockoutDuration, LockoutObservationWindow

# Find recently locked accounts
Search-ADAccount -LockedOut | Select Name, SamAccountName, LastLogonDate

# Unlock specific account
Unlock-ADAccount -Identity "john.smith"

# Monitor lockouts (Event 4740)
Get-WinEvent -FilterHashtable @{LogName='Security'; Id=4740} |
    Select TimeCreated, @{N='User';E={$_.Properties[0].Value}},
            @{N='LockedBy';E={$_.Properties[1].Value}} |
    Select-Object -First 20
```

## Brute Force Math

```
Without lockout (threshold = 0):
  Online brute force: 1,000 attempts/second
  8-char lowercase: 26^8 = 208 billion → ~2.4 days
  Common password list (10,000 passwords): 10 seconds

With threshold = 5:
  After 5 attempts → account locked for 15 minutes
  Effective rate: 5 attempts per 15 minutes = 20 attempts/hour
  10,000 common passwords: 500 hours = 20+ days
  → Online brute force completely impractical
```

## Windows 11 22H2+ Default Change

Microsoft changed the default lockout threshold from **0** (disabled) to **10** in Windows 11 22H2. PolicyForge recommends **5** for a stricter posture aligned with CIS.

## Balancing Security vs. Helpdesk Load

| Threshold | Security | Helpdesk Impact |
|---|---|---|
| 0 | ❌ No protection | ✅ No lockouts |
| 3 | ✅ Very strong | 🔴 High (typos, MFA fatigue) |
| **5** | **✅ Strong** | **🟡 Acceptable** |
| 10 | 🟡 Moderate | ✅ Low |
| 50+ | ❌ Weak | ✅ None |
